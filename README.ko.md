## 이 도구는 무엇인가요?

Google Gemini로 이미지를 생성하면 자동으로 워터마크가 추가됩니다. 이 워터마크는 이미지가 AI에 의해 생성되었음을 나타내는 반투명 레이블입니다.

이 도구는 **역 알파 블렌딩** 기술을 사용하여 Gemini 워터마크를 제거합니다. 이 기술은 워터마크 레이어 아래의 원래 픽셀 값을 수학적으로 복원합니다. 모든 처리는 브라우저 내에서 이루어지며, 이미지가 어떤 서버에도 업로드되지 않습니다.

## 사용 방법

### 1단계 — 도구 열기

모든 최신 브라우저(Chrome, Firefox, Safari, Edge)에서 [https://wr.imgplaceholdr.com](https://wr.imgplaceholdr.com)으로 이동하세요. 설치가 필요하지 않습니다.

### 2단계 — 이미지 업로드

업로드 영역을 클릭하거나 이미지를 드래그 앤 드롭하세요.

**지원 형식:** JPG, PNG, WebP  
**최대 파일 크기:** 이미지당 20MB  
**일괄 처리:** 지원됨. 여러 이미지를 동시에 업로드 가능

### 3단계 — 처리 완료 대기

브라우저가 몇 초 내에 자동으로 이미지를 처리합니다. 어떤 서버에도 전송되지 않습니다.

### 4단계 — 결과 미리보기

원본 이미지와 처리된 버전이 나란히 표시됩니다. 다운로드 전에 확대하여 품질을 확인할 수 있습니다.

### 5단계 — 깨끗한 이미지 다운로드

출력 형식을 선택하고 다운로드를 클릭하세요.

**사용 가능한 출력 형식:**

- PNG (최고 품질, 무손실)
- JPG (더 작은 파일 크기)
- WebP (웹 사용에 최적)

## 개인 정보 보호

모든 이미지 처리는 HTML5 Canvas API를 사용하여 브라우저 내에서 로컬로 이루어집니다. 이미지가 기기를 떠나지 않습니다. 파일을 수신하는 서버가 없으며 업로드한 내용에 대한 접근도 없습니다.
